수감자 파편 · 거울던전 · 인격 보관함

① 인격 보관함 탭

  • 수감자 섹션의 파편 수 입력칸에 현재 보유 파편 수를 입력하세요.

② 목표 인격 탭

  • 파편으로 뽑고 싶은 인격/에고 카드를 클릭해서 선택하세요. (복수 선택 가능)

③ 이 탭에서 설정 후 계산

  • 패스 보유 체크 — 패스 있으면 페스당 상자 3개, 없으면 1개
  • 목표 기간(일) 입력 — 기간 내 일퀘·주간퀘·일일 거울던전 상자를 자동 차감
  • 주간보너스 유형 선택 — 하드/노말 및 받는 방식에 따라 추가 상자 반영
  • 계산 → 버튼을 누르면 필요한 거울던전 추가 판수가 나와요.

계산 원리

  • 거울던전 1판 = 30XP = 3페스, 주간퀘 20XP/주, 일퀘 10XP/일
  • 주간 초기화는 매주 목요일 12시 기준으로 계산됩니다.

  • 파편으로 획득하고 싶은 인격 또는 에고 카드를 클릭해서 선택하세요.
  • 이미 보관함에 보유 중인 항목은 목록에 표시되지 않아요.
  • 선택하면 위에 수감자별 필요 파편·거울던전 요약이 바로 표시돼요.

보유 여부 체크

  • 보유로 체크한 항목은 목표 인격 탭에서 자동으로 제외됩니다.

파편 수 입력

  • 수감자 이름 옆 파편 수 입력칸에 현재 보유 파편을 입력하세요.
  • 입력한 파편 수는 파편/거울던전 탭 계산에 자동 반영됩니다.

데이터 저장

  • 모든 데이터는 이 기기 브라우저에 자동 저장돼요.
  • 다른 기기에서는 별도로 입력해야 합니다.
